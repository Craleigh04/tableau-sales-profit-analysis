# Tableau Sales & Profit Analysis

This project analyzes company profit trends using Tableau, focusing on why the Furniture category underperformed compared to Office Supplies and Technology.

## Overview

Using the Superstore dataset, this analysis explores multiple factors impacting profitability, including subcategories, regional performance, discounts, shipping methods, and product-level trends.

## Key Findings

- Furniture profits remained flat due to **high discounting and low margins**
- Discounts above ~20% significantly reduced profitability
- Certain subcategories (Tables, Bookcases) drove most losses
- A small number of products accounted for a large portion of negative profit
- Shipping costs, especially expedited shipping, reduced margins on large items
- Profitability varied significantly by region and location

## Why This Project Matters

This project demonstrates the ability to analyze business data, identify root causes of performance issues, and provide actionable recommendations using data visualization tools.

## Recommendations

- Limit discounts to 20% or less
- Remove or renegotiate low-performing products
- Optimize shipping methods for large items
- Adjust pricing strategies by region
